### What this site does

freelancer-templates.org is a static website hosted on GitHub Pages. It provides free Remotion video templates and a browser-based brand color customization tool.

### Data we process

**Hosting (GitHub Pages):** When you visit this site, GitHub's servers automatically log standard connection data (IP address, browser type, requested page, timestamp). This processing is necessary to deliver the website. GitHub's privacy statement applies: [docs.github.com/en/site-policy/privacy-policies](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement)

**Local browser storage:** The brand color customization tool can optionally save your color preferences to your browser's LocalStorage. This data never leaves your device and is not transmitted to any server. You can clear it at any time through your browser settings.

**Analytics (Plausible):** This site uses Plausible Analytics, a privacy-focused, cookie-free analytics tool. Plausible does not use cookies, does not collect personal data, and is fully GDPR compliant. All data is aggregated and no individual visitors can be identified. More information: [plausible.io/data-policy](https://plausible.io/data-policy)

**No accounts:** This site does not require registration or user accounts.

### Data processing on your device

The customize feature (logo upload, CSS parsing, color extraction) runs entirely in your browser. Uploaded logos and CSS files are processed locally using the Canvas API and JavaScript. No files are uploaded to any server.
