A 12-second portrait composition (1080×1920 · 360 frames) designed as an animated festival lineup announcement poster:

1. **WavyText headliner** — festival name enters with a bouncy spring scale, rendered as a wavy animated headline
2. **Date + location** — slides in from the left with a color accent bar
3. **2-column artist grid** — each name animates in from alternating sides with stagger (18 frames between each)
4. **Tickets CTA** — fades in last

The stock festival backdrop is set to 30% opacity with a heavy dark gradient overlay so the text is always legible.

## Color variants

**NeonGradient** — purple (`#7c3aed`) → pink (`#ec4899`) gradient overlay. High-energy electronic festival vibes.

**SunsetWarm** — orange (`#f97316`) → red (`#ef4444`) gradient overlay. Late-summer outdoor festival energy.

## Use cases

- Festival and event promoters sharing lineup reveals
- Creators hyping up events they're attending
- Music bloggers and entertainment pages announcing headliners
- Event ticketing social media content
