Adapted from [reactvideoeditor/remotion-templates](https://github.com/reactvideoeditor/remotion-templates). The glitch-text template was adapted to use theme accent colors for the RGB split layers instead of hardcoded cyan/magenta.

## Use cases

- Tech intro sequences
- Error or alert animations
- Cyberpunk-style branding
- Gaming content intros

## What you get

- **RGB split layers** - theme accent + accentSecondary as chromatic channels
- **Oscillating motion** - sine-wave driven glitch intensity
- **Screen blend** - mixBlendMode for authentic CRT effect
