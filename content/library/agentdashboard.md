Use when your audience needs to understand what an AI system does without reading code. The flow variant shows agent orchestration clearly.

## Use cases

- AI agent system architecture presentations
- Automation workflow demonstrations
- Team status boards for AI pipelines
- Product demos for AI-powered tools

## What you get

- **Control Panel** - main dashboard with agent status indicators
- **Flow** - agent-to-agent orchestration diagram
- **Matrix** - grid view of all agents with health and task counts
