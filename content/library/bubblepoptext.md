Adapted from [reactvideoeditor/remotion-templates](https://github.com/reactvideoeditor/remotion-templates). The bubble-pop-text template was adapted with themed gradients and our border-radius tokens. Each character pops in inside a styled bubble.

## Use cases

- Brand name animations for social media
- Short hook words (SALE, NEW, FREE)
- Playful intro sequences
- Logo text reveals

## What you get

- **Bubble letters** - each character in a rounded gradient container
- **Springy pop-in** - staggered scale animation per letter
- **Theme-aware** - borders, shadows, and gradients from theme tokens
