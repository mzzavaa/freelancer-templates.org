How to Use

From clone to rendered video in minutes

Three steps - whether you need a contract PDF or a thousand personalized videos.

1

Clone from GitHub

Fork or clone the repository. Browse the 54 template directories and pick what you need - document templates, Remotion compositions, or both.

git clone github.com/mzzavaa/
freelancer-templates.org
2

Customize props and styles

Edit composition props to match your brand - client name, colors, data. Document templates open in Word, Notion, or Google Docs. Remotion templates accept typed props.

3

Render with Remotion CLI

Render locally with the Remotion CLI, or scale to thousands of outputs with Remotion Lambda. One composition, unlimited unique videos.

npm install
npx remotion render MyComp out/video.mp4
