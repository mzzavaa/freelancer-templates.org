terminal
# 1. Install
npm install remotion @remotion/cli

# 2. Edit your composition props
import { TestimonialVideo } from './compositions';
const props = { name: "Jane Doe", quote: "...", brand: "#f59e0b" };

# 3a. Render locally with Remotion CLI
npx remotion render TestimonialVideo out/video.mp4

# 3b. Or render at scale with Remotion Lambda
npx remotion lambda render TestimonialVideo --props='{"name":"Jane"}'
How It Works

From template to deliverable in minutes

The same simple workflow whether you're filling a contract or rendering a thousand videos.

1

Choose a template

Browse by category or search for exactly what you need. Every template is production-ready and built for real freelance work.

2

Customize with your data

Fill in your details - client name, project scope, brand colors. Document templates open in Word, Notion, or Google Docs. Remotion templates accept typed props.

3

Export or render programmatically

Download your document instantly. For video, render locally or at scale with Remotion Lambda - one composition, thousands of unique outputs.

npm install && npx remotion render
-- or --
npx remotion lambda render
