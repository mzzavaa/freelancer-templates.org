4

Export CSS variables

Paste these into your project to apply your brand colors everywhere. Works with Remotion themes, CSS files, and any design system.

CSS 
:root {
  --brand-primary:   #1a365d;
  --brand-secondary: #2d3748;
  --brand-accent:    #4299e1;
  --brand-bg:        #f7fafc;
  --brand-text:      #1a202c;
  --brand-font:      'Inter', sans-serif;
}

Edit any value above or use the color pickers in Step 2 to update all variables at once.
